# スクリプト言語を触ってみよう

## どの言語？

このコースでは、スクリプト言語としてRubyとJavascriptを採用します。

## 何をやるの？

C言語とRuby,Javascriptのコードを比較しながら、実際に実行してもらいその違いを覚えて貰います。

## こんな感じ

10回ループする処理

### in C

    !c
    for(int i = 0; i < 10; ++i) {
        printf("%d\n", i);
    }

### in Ruby

    !ruby
    10.times do |idx|
      puts(idx)
    end
