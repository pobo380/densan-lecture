main関数が不要 *

in C

#include "stdio.h"

int main(int argc, char *argv[]) {
    printf("Hello World!"); // comment
    return 0;
}

プログラムの始まりを示すためにmain関数を定義する必要がある。

in Ruby

printf("Hello World!") # comment

プログラムはファイルの1行目から実行される。main関数を定義する必要はない。

セミコロンが不要 *

in C

#include "stdio.h"

int main(int argc, char *argv[]) {
    printf("Hello ");
    printf("World!");

    printf("Hello "); printf("World!");

    return 0;
}

セミコロン(;)が文の終わりを示す。

in Ruby

printf("Hello ")
printf("World!")

printf("Hello "); printf("World!")

改行が文の終わりを示す。セミコロンを使うことも出来る。

コメントの書き方

in C

// がコメント
/*
  これもコメント
  複数行に渡る記述が可能
*/

in Ruby

# がコメント
=begin
  これもコメント
  複数行に渡る記述が可能
  =begin, =endは行頭に記述される必要があります。
=end

基本的には記号が違うだけ。=begin, =endは行頭に記述しなければいけない点に注意。

変数の宣言が不要 *

in C

void main() {
    int a, b, c;
    a = 10;
    b = 2;
    c = a + b; // => 12
    d;         // Compile Error!!
}

変数を利用する場合は、事前に宣言をする必要がある。宣言をしていない変数を参照しようとするとエラーとなる。

in Ruby

a = 10
b = 2
c = a + b # => 12
d         # => nil

宣言が不要。代入した時点で変数が定義される。定義されていない変数(d)を参照した場合はnil(CでいうNULLのようなもの)となる。

変数に型が無い *

in C

void main() {
    int var;
    var = 10;       // => 10  OK
    var = 0.33524;  // => 0
    var = "foobar"; // Compile Error!!
}

int型として宣言した変数varに、文字列を代入することは出来ない。

in Ruby

var = 10
var = 0.33524      # => 0.33524      OK 
var = "foobar"     # => "foorbar"    OK
var = [5, 4, 6, 2] # => [5, 4, 6, 2] 配列でもOK

変数に型が無いため、どんな値でも代入することが出来る。

定数の宣言方法

in C

#include "stdio.h"

#define SCREEN_WIDTH  800 // 定数の宣言
#define SCREEN_HEIGHT 600 //

void main() {
    printf("width: %d, height: %d", SCREEN_WIDTH, SCREEN_HEIGHT);
}

defineマクロによって定数を定義する。コンパイル時にマクロ定数への参照が値へ置き換えられる。マクロ名が大文字である必要はない。

in Ruby

SCREEN_WIDTH  = 800 # 定数の宣言
Screen_height = 600 # 定数
screen_fps    = 40  # 変数

printf("width: %d, height: %d", SCREEN_WIDTH, Screen_height);

変数名の頭を大文字にすると'変更不可の変数'となる。

if文の括弧が不要

in C

#include "stdio.h"
void main() {
    if(4 < 2) {
        printf("true");
    }
    else {
        printf("false");
    }
}

in Ruby

if 4 < 2 
  puts("true")
else
  puts("false")
end
# if 条件
#   ...
# end

中括弧ではなくif〜endで表す。条件文にも括弧が要らない。これはwhileでも同じ

else ifではなくelsif

in C

#include "stdio.h"
void main() {
    if(4 < 2) {
        printf("4 < 2 is true.");
    }
    else if(4 > 2) {
        printf("4 > 2 is true.");
    }
}

in Ruby

if 4 < 2 
  puts("4 < 2 is true.")
elsif 4 > 2
  puts("4 < 2 is true.")
end

表記が異なる。

関数の宣言方法 *

in C

int foo(int x, int y) {
    return x * y;
}
// 戻り値の型 関数名(引数型 引数名, ...) {
//     return 戻り値;
// }

in Ruby

def foo(x, y) 
  x + y
end
# def 関数名(引数名, ...)
#   戻り値
# end

中括弧を用いない。また、returnを省略することが出来る。

文字列の長さを気にしなくてよい *

in C

#include "stdio.h"

#define BUFF_LENGTH 5

void main() {
    char buff[BUFF_LENGTH];

    fgets(buff, BUFF_LENGTH, stdin);
    printf("%s\n", buff);
}
// input:  12345678\n
// output: 1234\n

in Ruby

buff = STDIN.gets

printf("%d", buff)
# input:  12345678\n
# output: 12345678\n

Rubyでの文字列は、文字の配列ではない。例えば"c"は、長さ1の文字列である。

#include "string.h"

文字列の一部を取り出すのが簡単 *

in C

#include "stdio.h"
#include "string.h"

#define BUFF_LENGTH 20

void main() {
    char str[BUFF_LENGTH] = "foofoo";

    strncpy(&(str[3]), "bar", 3);
    printf("%s\n", str);
}

in Ruby

str = "foofoo"

str[3, 3] = "bar" # 4文字目から 3文字分置き換え
printf("%s\n", str)

str[0..2] = "bar" # 1文字目から3文字目まで 置き換え
printf("%s\n", str)

それからそれから

文字列を整数に変換する

str = "666"

val = str.to_i + 444
printf "%d", val

文字列中の式を評価して値を展開する

def nibai(x)
  x * 2
end

value = 205

puts "value is #{value}" # => "value is 205"
puts "value is #{nibai(value)}" # => "value is 410"

#{...}で囲った式は展開され、文字列に埋め込まれる。関数を呼び出すことも可能。

文字列の終わりの改行を取り除く

str = "pobo kawaii\n"
str.chomp!

print str # => "pobo kawaii"
